# mobile_learning
一个简单的基于Android的移动学习平台，具有课程选择、资源下载、上传作业、在线测试、在线交流以及修改信息模块。服务器采用Jsp+Servlet模式，具有相应的后台界面。请使用android3.0及以上版本导入客户端，以及低于8.0版本MySql进行连接。（这是本人边学边做的项目，还有很多需要修改的地方，但鉴于修改太过麻烦，因此就只有这个版本）
